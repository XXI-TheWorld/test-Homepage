## Context

当前 LinkPage 已实现日式极简风格（白色背景、Noto Sans JP 字体、樱花花瓣飘落）。用户希望进一步强化日式美学：
1. 樱花花瓣更大更显眼
2. 链接按钮添加木纹边框
3. 头像添加金色边框（武士头盔风格）

## Goals / Non-Goals

**Goals:**
- 增大樱花花瓣尺寸至 20-35px
- 使用 CSS 渐变实现木纹边框效果
- 头像金色边框呈现武士头盔装饰感
- 保持页面整体极简风格不受破坏

**Non-Goals:**
- 不改变页面布局结构
- 不修改樱花动画的运动轨迹
- 不添加任何新的交互功能

## Decisions

### 1. 樱花花瓣尺寸增大
- **当前**：8-15px
- **新尺寸**：20-35px
- **理由**：更大的花瓣更显眼，视觉冲击力更强
- **备选**：保持现有尺寸仅增加花瓣数量（已拒绝，太多花瓣影响阅读）

### 2. 木纹按钮边框
- **技术方案**：CSS 多层线性渐变叠加
- **渐变角度**：90度（水平木纹）
- **配色**：
  - 底色：`#8B7355`（棕色）
  - 纹理：`#6B5344`（深棕）至 `#A08060`（浅棕）
- **边框宽度**：2px
- **实现**：
  ```css
  border: 2px solid transparent;
  background:
    linear-gradient(#fff, #fff) padding-box,
    linear-gradient(90deg, #6B5344, #8B7355, #A08060, #8B7355, #6B5344) border-box;
  ```

### 3. 金色头像边框
- **颜色**：`#D4AF37`（金色）
- **风格**：实线边框，非渐变，呼应日本武士头盔装饰
- **边框宽度**：3px
- **实现**：
  ```css
  border: 3px solid #D4AF37;
  ```
- **备选方案**：
  - 双层边框（内金外黑）→ 拒绝，过度装饰
  - 金色渐变 → 拒绝，实线更接近盔甲金属质感

## Risks / Trade-offs

| 风险 | Mitigation |
|------|------------|
| 木纹边框与白色背景对比不足 | 调整木纹配色确保可见性 |
| 金色边框过于突兀 | 使用饱和度适中的金色，不过分闪亮 |
| 大花瓣影响内容阅读 | 限制花瓣数量（15-18片），保持 pointer-events: none |

## Open Questions

- 木纹渐变是否需要垂直变化以更真实？
- 金色边框是否需要微妙的内阴影增加立体感？
