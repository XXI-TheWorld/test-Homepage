## Context

当前 LinkPage (`index.html`) 采用黑底绿色 "Matrix" 代码风格，带有发光效果和闪烁动画。用户希望转型为日式极简风格——白色背景、大量留白、如同日本神社的宁静感。同时需要保留链接按钮的 hover 变色效果，并新增樱花花瓣飘落动画。

## Goals / Non-Goals

**Goals:**
- 白色背景，营造纯净、宁静的视觉效果
- 使用无衬线字体（Noto Sans JP），提升跨语言美感
- 扩大元素间距，增强垂直节奏和呼吸感
- 保留链接按钮的 hover 变色效果（简约风格，非发光）
- 实现樱花花瓣飘落动画，增添日式氛围

**Non-Goals:**
- 不移除现有功能，只改变视觉呈现
- 不改变页面结构或链接配置
- 樱花动画保持轻柔，不影响内容阅读

## Decisions

### 1. 字体选择：Noto Sans JP
- **选择理由**：Google Fonts 提供，支持日语汉字与假名，与无衬线体美学高度契合
- **备选**：Inter（更西式）、Roboto（过于通用）
- **最终选择**：Noto Sans JP，权重使用 300（轻）和 400（常规）

### 2. 配色方案
| 元素 | 颜色 |
|------|------|
| 背景 | `#FFFFFF`（纯白） |
| 主文字 | `#1A1A1A`（近黑，低对比度更柔和） |
| 副文字 | `#666666`（灰，bio 和 prompt） |
| 按钮边框 | `#E0E0E0`（浅灰） |
| 按钮 hover | `#F5F5F5`（极浅灰背景） |
| 按钮 hover 文字 | `#1A1A1A` |

### 3. 间距系统
- 页面 padding：48px（移动端 24px）
- 容器 max-width：480px
- 元素间距：32px（header 到 links）
- 按钮间距：12px
- 按钮高度：52px（略高，强调垂直线条感）

### 4. 樱花动画实现
- **技术方案**：纯 CSS + 少量 JavaScript 生成花瓣元素
- **花瓣数量**：15-20 片同时飘落
- **动画参数**：
  - 飘落时长：8-15 秒（随机）
  - 横向摆动：sin 波模拟风效果
  - 花瓣尺寸：8-15px
  - 旋转：轻微随机旋转
- **花瓣颜色**：`rgba(255, 182, 193, 0.6)`（半透明粉色）
- **花瓣形状**：CSS clip-path 或 SVG 椭圆

### 5. 组件样式

**头像**
- 白色背景，浅灰边框
- 无阴影、无发光

**按钮**
- 纯白背景，浅灰细边框
- 无圆角（方形边角更接近神社鸟居感）
- hover：背景变为极浅灰，文字变为深灰
- 无动画过渡，hover 效果即时切换

## Risks / Trade-offs

| 风险 |  Mitigation |
|------|-------------|
| 日文字体加载慢 | 使用 `font-display: swap`，提供回退字体 |
| 樱花动画性能问题 | 使用 `will-change: transform`，限制花瓣数量 |
| 花瓣遮挡可点击区域 | 设置 `pointer-events: none`，动画层置于最低 |
