## Context

当前评论系统游客可以删除自己的评论，但无法删除他人评论。需要一个简单的前端管理员模式来管理内容。

## Goals / Non-Goals

**Goals:**
- 通过激活暗号开启管理员模式
- 管理员可删除任意评论
- 管理员名字显示为"管理员"并使用特殊样式

**Non-Goals:**
- 不实现用户权限分级
- 不持久化管理员状态（仅当前浏览器）
- 不支持多管理员

## Decisions

### 1. 管理员激活
在评论区添加"身份"区域，点击"管理员登录"按钮，输入暗号验证。验证成功后保存 `admin_logged_in: true` 到 localStorage。

### 2. 激活暗号
使用 "admin123" 作为默认暗号。激活后不再显示登录按钮。

### 3. 评论渲染修改
检查 localStorage 中 `admin_logged_in` 状态：
- 如果是管理员：每条评论显示红色 [删除] 按钮
- 如果是普通用户：只显示自己评论的删除按钮

### 4. 管理员样式
"管理员"文字使用红色（#e53935）加粗显示。

## Risks / Trade-offs

- **本地存储风险**: 任何知道暗号的人都可以激活管理员，这是可接受的风险（本地管理）
