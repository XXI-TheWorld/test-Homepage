## Context

LinkPage 当前具有日式极简风格（白色背景、木纹按钮、金色头像边框、樱花飘落动画）。用户需要增加以下功能：复制微信号、深色模式切换、访问计数器、背景音乐播放。

## Goals / Non-Goals

**Goals:**
- 实现微信号一键复制功能，带"已复制"反馈提示
- 深色/浅色模式切换（右上角），带平滑过渡
- localStorage 访问计数器
- 音乐播放按钮（播放/暂停，默认静音）

**Non-Goals:**
- 不实现用户自定义微信号功能（硬编码）
- 不提供音乐选择功能（单一背景音乐）
- 不实现访问数据分析

## Decisions

### 1. 复制微信号功能
- **技术方案**：使用 `navigator.clipboard.writeText()` API
- **备选**：`document.execCommand('copy')`（已废弃）
- **用户反馈**：在按钮位置显示 "已复制 ✓" 提示，2 秒后自动消失
- **微信号**：硬编码为 `WeChatID`（需确认实际值）

### 2. 深色模式切换
- **配色方案**：
  - 浅色模式：当前样式（白底黑字）
  - 深色模式：背景 `#1A1A1A`，文字 `#F5F5F5`，边框 `#333333`
- **实现**：CSS 变量 + `data-theme` 属性在 body 上
- **切换按钮**：右上角固定位置，太阳/月亮图标切换
- **过渡动画**：CSS `transition: background-color 0.3s, color 0.3s`

### 3. 访问计数器
- **存储**：`localStorage.getItem/setItem('visitCount')`
- **显示**：页面底部居中，"访问次数: X"
- **逻辑**：每次页面加载时 +1

### 4. 背景音乐
- **实现**：HTML5 `<audio>` 元素
- **音乐源**：使用公开可用的背景音乐 URL（需用户提供或使用 placeholder）
- **控制**：页面右下角小图标按钮，点击切换播放/暂停
- **默认状态**：静音 (`muted`)
- **图标**：播放▶️ / 暂停⏸️

### 5. 外链新标签打开
- 所有 `target="_blank"` 的链接已存在于 HTML 中
- 确认 mailto 链接不需要新标签打开

## Risks / Trade-offs

| 风险 | Mitigation |
|------|------------|
| 剪贴板 API 在某些浏览器受限 | 提供 fallback 提示"复制失败，请手动复制" |
| 用户无音乐文件 | 使用公开 placeholder URL 或内嵌短音频 |
| 深色模式样式遗漏 | 逐一检查所有颜色相关 CSS 变量 |
| localStorage 不可用 | 静默降级，计数器不显示 |
