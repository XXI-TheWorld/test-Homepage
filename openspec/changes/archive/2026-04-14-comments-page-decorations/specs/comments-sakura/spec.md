## ADDED Requirements

### Requirement: Sakura petal animation on comments page
The system SHALL display falling sakura petal animation on the comments page, identical to the main page experience.

#### Scenario: Display sakura petals
- **WHEN** user navigates to the comments page
- **THEN** system displays 15-20 sakura petals falling from the top of the viewport
- **AND** petals have natural sway animation
- **AND** petals have varying sizes (20-35px)
- **AND** petals fade out as they reach the bottom
